# Playbook: Install Notepad Test

## Цель
Тестовая задача для проверки системы плейбуков: открыть Notepad, написать текст, сохранить файл и закрыть.

## Переменные
- AppName: "Notepad"
- TestText: "Hello from xelthAGI Playbook System!"
- SaveFileName: "test_playbook.txt"

## Шаги выполнения (SOP)

### 1. Запуск приложения
- **Действие**: Открыть меню Пуск (Start Menu)
- **Действие**: Найти и запустить приложение "Notepad"
- **Проверка**: Убедиться, что окно с заголовком "Untitled - Notepad" или "Notepad" открылось

### 2. Ввод текста
- **Действие**: Кликнуть в текстовое поле (основная область окна)
- **Действие**: Ввести текст: `{TestText}`
- **Проверка**: Убедиться, что текст появился в окне

### 3. Сохранение файла
- **Действие**: Нажать комбинацию Ctrl+S ИЛИ выбрать File → Save
- **Действие**: В диалоге "Save As" ввести имя файла: `{SaveFileName}`
- **Действие**: Нажать кнопку "Save"
- **Проверка**: Убедиться, что заголовок окна изменился на "{SaveFileName} - Notepad"

### 4. Закрытие приложения
- **Действие**: Нажать кнопку "Close" (X) в правом верхнем углу окна
- **Проверка**: Убедиться, что окно Notepad закрылось

## Примечания
- Если на шаге 3 не видно кнопку "Save" в текстовом дереве, используйте команду `inspect_screen` с качеством 50% для визуального поиска.
- Этот плейбук предназначен для тестирования базовой функциональности системы.
